Using QML TestCase

QML TestCase Components are used to define the individual tests that need to be run for a QML Component.

A TestCase should reside in a tst_ prefixed QML file.

A TestCase will hold one or more Javascript test functions that will be the actual tests run. Each of these test functions will have a test_ prefix

A Simple Test

Testing a components background color is a simple example of how to use the TestCase Component and some of the built in test functions it offers.

First define a Background.qml:

import QtQuick 2.0
Rectangle{
    id:background
    width:100
    height:100
    color:"black"
}

Next define a tst_background.qml and add the following code:

import QtQuick 2.0
import QtTest 1.2
TestCase{
    id:backgroundTests
    name:"TestBackground"
}
