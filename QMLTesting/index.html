QML Testing Overview

QML Testcase is a helpful module in QT that allows for developers to create Automated UI tests for QT Applications. Sadly the official QT documentation, and the internet as a whole is somewhat lacking on helpful documentation on setting up and using QML Testcase to actually test applications. My goal is to document some of the ways I have been using QML Testcase, and slowly work on building out some tutorials for its usage.
