Think!

Think! was one of the first completely Computer Science-based exhibits that was offered to our library partners. It is designed as an introduction into computer logic using combination logic. It runs on a Arduino Mega board that uses LED buttons and lights to display which sentences correspond to different combinations of buttons.

It was laser-engraved white acrylic that was then painted to create contrast. The buttons were originally fitted with 12V incandescent bulbs that I converted to 5V LED bulbs to allow for much easier control by the Arduino Mega.
